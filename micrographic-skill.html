Micrographic Skill

My product for AI coding agents to generate dense, technical, schematic UI.

Overview

Product Design System Open Source

Micrographic Skill provides a complete style system for code-generation agents. It pushes UI output toward an industrial, spec-sheet aesthetic using compact typography, monochromatic palettes, and structured annotation patterns.

The skill is designed for product cards, dashboards, docs, and interfaces that need high information density with strong visual hierarchy.

Install

Quick install

npx micrographic-skill

For Cursor projects, this installs to .cursor/skills/micrographic/SKILL.md.
